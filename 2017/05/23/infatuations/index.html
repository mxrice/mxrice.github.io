infatuations

因为迷恋，所以进步。

因为迷恋，所以有了这个blog.
在公司闲着无聊的时候，总想折腾点什么，因为前几天提到了一个朋友去学js,于是萌生了自己用github+hexo搭建自己博客的想法。使用github的原因一个是因为国内的各种博客基本处于死掉或者半死不活的状态，大家的使用频率都太少了，github的受众大，免费且稳定。过年的时候把笨重的AW搬回了家里，身边只留下当pad使用的mbp。于是用osx来写，但是网上的教程和日志大多都是旧时的windos，跟着老旧的步骤，自然问题多多。
步骤：
首先要装好hexo的环境： node.js+github
装node： 为了避免因为系统差异的原因下载出现报错，去node的官网选择了mac os的版本下载安装
装git： 原因同上，选择了官方下载安装包安装git
npm：npm在node的安装包顺带有
安装hexo：
在终端使用命令安装：npm install hexo -g
初始化文件夹：hexo init
在初始化的时候出现了报错，无法找到hexo
排查环境： node -v
git
npm -v
均有版本信息，说明环境安装没有问题，文件路径均为英文，来回折腾了一下午，还是报错找不到hexo。根目录下有hexo文件说明安装hexo是成功的。因为下班所以放弃了
第二日尝试了“重启”，把node和git全部重新安装，依然报错，一度要放弃的时候。在安装git的时候见到了一句话：”xcode自带git“。考虑git安装问题，清除了之前的node、git、npm、nvm决定下载xcode。
下载安装Xcode：app store 直接点击下载
下载安装时提示系统版本过低，此时系统版本为10.11 提前下班跑到天河城apple店花228买到了Thunderbolt转以太网的转接线
晚上更新系统至最新：10.12.5 在app store点击下载xcode提示还需要4G空间， xcode大小为3G左右，此时系统空余空间为11G 故xcode安装环境大致为15G左右
安装Homebrew：/usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
安装Node.js：brew install node
测试node环境：在根目录新建了一个js文件：”hello”
var http = require(‘http’);
http.createServer(function(req, res){
res.writeHead(200, {‘Content-Type’: ‘text/plain’});
res.end(‘Hello World\n’);
}).listen(8808, ‘127.0.0.1’);
console.log(‘Server running at http://127.0.0.1:8808‘);
终端输入：node hello.js 会生成一个本地界面点击在浏览器可以看到”hello world”字样
